# Doraku

This repositry is source code of Doraku.  
Application URL: [https://doraku.netlify.com](https://doraku.netlify.com)

## How to run in local

1. Run backend server

    ```bash
    cd backend
    go build
    ./doraku
    ```

2. Run front end

    ```bash
    cd frontend/doraku
    npm start
    ```

3. Access to frontend
    URL: [http://localhsot:3000](http://localhsot:3000)
